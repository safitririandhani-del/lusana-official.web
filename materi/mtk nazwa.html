POSTER MATEMATIKA

Dibuat Oleh: Nazwa

Cara penyelesaian

x = Tas
y = Lampu hias dari sendok plastik
z = Massroom lamp

Sistem persamaan:
1. x + 2y + 2z = 170.000
2. 2x + 3y + z = 205.000
3. 2x + 4y + 2z = 180.000

--- • Eliminasi Persamaan (2) dan (3)

Kalikan persamaan (1) dengan 2:
2x + 4y + 4z = 340.000
Kurangkan persamaan (2):
(2x + 4y + 4z) – (2x + 3y + z) = 340.000 – 205.000
y + 3z = 135.000 …(A)
Kurangkan persamaan (3) dari hasil di atas:
(2x + 4y + 4z) – (2x + 4y + 2z) = 340.000 – 180.000
3y + 2z = 160.000 …(B)

--- • Eliminasi Persamaan (A) dan (B) untuk mencari z

Kalikan (A) dengan 3:
3y + 9z = 405.000
Kurangkan dengan (B):
(3y + 9z) – (3y + 2z) = 405.000 – 160.000
7z = 245.000
z = 245.000 / 7
z = 35.000

--- • Substitusi z ke Persamaan (A) untuk mencari y

y + 3(35.000) = 135.000
y + 105.000 = 135.000
y = 135.000 - 105.000
y = 30.000

• Substitusi y dan z ke Persamaan (1) untuk mencari x

x + 2(30.000) + 2(35.000) = 170.000
x + 60.000 + 70.000 = 170.000
x + 130.000 = 170.000
x = 170.000 - 130.000
x = 40.000

Hp = {(40.000, 30.000, 35.000)}
