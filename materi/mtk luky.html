POSTER MATEMATIKA

Dibuat Oleh: Adilla

Cara Penyelesaian

x = Tas
y = Rak
z = Lampu

1. Kita memiliki sistem persamaan linear tiga variabel:

x + y + z = 110.000 …(1)
x + y + 2z = 140.000 …(2)
x + 2y + 2z = 180.000 …(3)

2. Eliminasi (1) dari (2) untuk mendapatkan z

(x + y + 2z) − (x + y + z) = 140.000 − 110.000
z = 30.000

3. Substitusi nilai z = 30.000 ke persamaan (1)

x + y + 30.000 = 110.000
x + y = 110.000 - 30.000
x + y = 80.000 …(4)

4. Substitusi nilai z = 30.000 ke persamaan (3)

x + 2y + 2(30.000) = 180.000
x + 2y + 60.000 = 180.000
x + 2y = 180.000 - 60.000
x + 2y = 120.000 …(5)

5. Kurangkan persamaan (4) dari persamaan (5)

(x + 2y) − (x + y) = 120.000 − 80.000
y = 40.000

6. Substitusi nilai y = 40.000 ke persamaan (4)

x + 40.000 = 80.000
x = 40.000

Hp = {(40.000,40.000,30.000)}

Harga per barang

Rak: Rp.40.000
Tas: Rp.40.000
Lampu hias dari sendok plastik: Rp.30.000
